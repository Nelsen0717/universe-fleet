# 裝備發放 v4 —「暗艦點燈」：沉浸重設計規格

**日期**：2026-07-06 ｜ **前情**：v3 結構對（單線/單狀態/嚮導、Nelsen 認可「雜訊少」）、但**體驗像 PPT**（Nelsen 原話）。
**病根診斷（一句話）**：把提案頁的設計語言（白底 hairline 極簡）錯套到遊戲上——那是文件的西裝、不是遊戲的世界。v4 保留 v3 的骨架、換掉全部的皮與血。

## 一、世界觀即進度條：「暗艦點燈」

**核心意象**：你登上一艘沉睡的暗艦。艦橋幾乎全黑、只有你面前一盞待命燈。**每完成一關、艦上一個系統真的亮起來**——儀表板亮、艦員站位燈亮、引擎嗡鳴聲加厚一層。五關完成 = 整座艦橋燈火通明、引擎滿鳴、出航。
**為什麼這是對的**：進度不再是「第 X／5 關」的字、是**你親眼看著船活過來**——沉浸感與清晰度同一個機制、不再打架。

## 二、視覺語言（推翻 v3、遊戲自己的衣服）

- **基調**：深空藍黑（#0a0e1a 系）艦橋、暖琥珀儀表光（#e8a34c 系）、冷青操作光（#7fd4e0 系少量）——光是配色的主角、顏色是光的載體
- **質感**：儀表面板細噪點、微掃描線、按鈕按下有光暈回饋；星窗外星野緩慢視差（Canvas、沿用但改深空版）
- **既有資產的深色化**：墨線插畫 CSS `filter: invert()` 反白成「星圖線稿」（白線在深空上發微光——比原白底更像這個世界的東西）；mentor-portrait（全彩）放進「通訊視窗」框（邊緣光暈＋輕微電波紋動畫）——彩色的人像在深色艦橋裡 = 唯一的人、視覺語意剛好
- **字體**：介面等寬（儀表感）、對話用舒適黑體；禁 PPT 式置中大標排版——資訊都長在「儀表、面板、通訊窗」這些世界物件上

## 三、聲音（沉浸的一半、全部 WebAudio 合成、無音檔）

- **環境層**：低頻艦橋嗡鳴（常駐、極低音量、每點亮一關疊厚一層——聲音也是進度條）
- **回饋層**：按鍵短嗶、複製成功「喀」、系統點亮上升和弦＋繼電器「咚」
- **過場層**：關與關之間 0.5 秒無線電雜訊沙沙——老師通訊接入的儀式
- 預設開、顯眼靜音鍵、`prefers-reduced-motion` 連動減效

## 四、老師 = 通訊視窗（人味的來源）

Nelsen 老師不再是「名牌＋文字」——是艦橋左上的**通訊視窗**：頭像（真圖）＋說話時輕微光暈呼吸＋逐字打出＋（合成）短促通訊開頭音。每關開始他說一句引導、完成說一句過場（沿用 v3 台詞、已驗證是人話）、卡關超過 90 秒他主動說一句鼓勵（計時觸發、僅一次、不煩人）。

## 五、關卡 = 控制台面板（高擬真三件套的正確歸宿）

每一關是艦橋的一個**控制台特寫場景**（鏡頭從艦橋全景推近到該面板）：
1. **面板頂**：本關目標一句（老師通訊窗同步說）
2. **面板主體 = 高擬真三件套**（這是 v3 遺失的、從 onboarding 原樣取回並深色化）：
   - 真實畫面模擬：macOS 終端機殼（深色、紅黃綠鈕、% 提示符、指令原文）／Claude Code 輸入列復刻／瀏覽器網址列——嵌在控制台裡、像面板上的「外部螢幕監看畫面」、標「畫面模擬」小字
   - 你做的動作：一步一動、動詞開頭、**複製鍵是面板上的實體感按鈕**（按下有光暈與喀聲、真的進剪貼簿）
   - 你應該看到：結果比對畫面（終端機吐什麼、Claude Code 回什麼）
3. **完成儀式**：按「回報完成」→ 面板跑 1.5 秒「訊號偵測」掃描動畫 → 對應系統亮起 → 鏡頭拉回艦橋全景看那盞燈亮（每次 2 秒、可跳過）→ 艦員甦醒通訊窗一句（沿用 v3 台詞）→ 下一關
4. 通關碼機制原樣保留（教練真驗證、不動）

## 六、結尾（情緒收口）

第五關完成 → 全艦點燈序列（燈一排排亮、嗡鳴推滿、星窗外引擎光）→ 五艦員通訊窗依序一句 → 老師：「艦活了。開課那天、帶著它來。」→ 證書（深色金框版、名字發光）。全程 ≤ 20 秒、可跳過。

## 七、不變的（v3 驗證過的骨架、一字不動）

單線流程／單一狀態鍵 `fleetOnboard.v2`（沿用、加 `litSystems[]`）／帶 token 鎖的轉場／計時器 teardown 契約／老師台詞內容／五關內容與白名單連結／通關碼／回訪續關／手機直式優先／「絕不重犯」七條全部繼承。

## 八、驗收（v4 特有、疊加 v3 驗收之上）

1. **靜音截圖測試**：任抽三個畫面給零脈絡者看、能說出「這是個（太空）遊戲」而不是「這是簡報」
2. 每關完成後、艦橋全景可見**對應系統真的亮了**（截圖對比第 0 關 vs 第 3 關）
3. 聲音分層實測：環境嗡鳴隨進度疊厚（錄前後對比或以 gain 值 stdout 佐證）；靜音鍵全域生效
4. 三件套高擬真回歸：終端機殼／Claude Code 輸入列／「你應該看到」三者每關齊備（grep＋截圖）
5. 手機 390 直式：控制台面板單欄堆疊、光效不吃效能（維持流暢）；`prefers-reduced-motion` 下光效降級但流程完整
6. v3 全部驗收重跑（連點、續關、疊字防護、白名單）
